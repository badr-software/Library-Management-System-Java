# 📚 Library Management System (Java & OOP)

A comprehensive university project designed to manage library operations efficiently. This system was built using **Java** and fundamentally applies **Object-Oriented Programming (OOP)** core principles.

---

## 🛠️ OOP Concepts Applied:
* **Encapsulation:** Securing book records and user data by utilizing private attributes with explicit getters and setters.
* **Inheritance:** Derived specific user roles (e.g., Students, Staff) from a foundational base `User` class to reduce code redundancy.
* **Polymorphism:** Overriding methods to customize search results and borrowing permissions dynamically based on the account type.
* **Abstraction:** Defined essential blueprints for library actions to separate high-level operations from system implementations.

---

## 💾 Data Management & Storage:
The project seamlessly reads and updates permanent records including book logs, active borrowing transactions, and user credentials across files and local databases (`.txt` & `.db`), ensuring data persistence between sessions.
